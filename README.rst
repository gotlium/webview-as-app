WebView as App
==============

Installation:
-------------

**OS X:**

.. code-block:: bash

    $ brew install pyqt
    $ wget https://raw.githubusercontent.com/gotlium/webview-as-app/master/main.py


**Linux:**

.. code-block:: bash

    $ apt-get install python-qt4
    $ wget https://raw.githubusercontent.com/gotlium/webview-as-app/master/main.py


Usage:
------

.. code-block:: bash

    $ python main.py -u https://www.google.com/ -i ~/Pictures/google.png -n 'Google Search'
